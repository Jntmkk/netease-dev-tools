1. 引言
[公司名称]（以下简称“我们”）非常重视用户的隐私和数据安全。本隐私政策旨在向您说明 [插件名称]（以下简称“本扩展”）如何收集、使用、存储和保护您的信息。
本扩展是专为 [公司名称] 内部员工设计的数据库管理辅助工具。通过使用本扩展，您确认您是 [公司名称] 的授权员工或合作伙伴，并同意本政策所述的数据处理实践。
2. 我们收集的信息
为了提供内部数据库查询与管理功能，本扩展会收集以下类型的信息：
2.1 个人身份信息 (Personal Identifiable Information)
员工标识：为了识别您的身份并记录操作日志，我们可能会收集您的员工工号、企业内部邮箱或姓名。
收集方式：通过企业内部单点登录（SSO）系统或您手动输入的身份凭证。
2.2 身份验证信息 (Authentication Information)
访问令牌 (Tokens)：为了发起安全的内部接口请求，扩展会临时存储会话令牌（Session Token）或 API Key。
凭证：用于验证您有权访问内部数据库系统的凭据。
注意：我们不会存储您的明文密码。密码仅在登录验证过程中通过加密通道传输。
2.3 本地存储数据 (Local Storage Data)
配置信息：为了提升使用体验，扩展会使用 Chrome 的 chrome.storage API 在您的本地浏览器中存储偏好设置、最近查询记录或缓存数据。
数据隔离：这些存储数据仅限于您的本地浏览器环境，其他网站或扩展无法访问。
3. 信息的使用目的
我们收集上述信息仅用于以下特定目的：
身份验证：验证您是否为授权员工，确保只有内部人员可以访问数据库工具。
服务功能：发起网络请求以查询、展示或管理公司内部数据库内容。
安全审计：记录操作日志以满足公司内部安全合规要求（如：谁在什么时间查询了哪些数据）。
功能优化：保存您的本地配置，以便下次使用时无需重复设置。
4. 信息的共享与披露
内部共享：您的身份信息及操作日志可能会在 [公司名称] 内部的安全系统、数据库审计系统中共享，用于安全监控和权限管理。
不向第三方出售：我们不会向任何外部第三方出售、出租或交易您的个人信息。
法律要求：除非法律法规要求或为了配合公司内部安全调查，否则我们不会向外部实体披露数据。
5. 数据安全与存储
传输加密：所有网络请求均通过 HTTPS 加密协议传输，防止数据在传输过程中被窃取。
本地存储：敏感的身份验证信息（如 Token）会存储在 Chrome 的安全存储区域，并尽可能设置有效期，过期自动失效。
访问控制：只有经过授权的公司内部服务器才能接收和处理扩展发送的数据。
6. 您的权利
作为用户，您拥有以下权利：
访问与更正：您可以通过公司内部系统访问或更正您的个人身份信息。
清除数据：
您可以在扩展设置中点击“清除缓存”或“退出登录”来删除本地存储的敏感信息。
卸载本扩展将自动清除所有存储在本地的扩展数据。
撤回同意：如果您不再希望使用本工具，请停止使用并卸载扩展，或联系公司 IT 部门收回您的访问权限。
7. 儿童隐私
本工具仅面向 [公司名称] 的成年员工使用，不面向 18 岁以下的儿童。我们不会故意收集儿童的个人信息。
8. 政策变更
我们可能会不时更新本隐私政策。更新后的政策将发布在本页面上。对于重大变更，我们会通过公司内部通知或扩展内弹窗的形式告知您。
